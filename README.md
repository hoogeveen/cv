## CV Generator
Create your own CV in a couple of seconds by filling out a JSON file.

### How to get started
 - Clone this repository to your machine.
 - Run `npm install`
 - Run `npm start:dev`
 - Profit.
 
The design of this CV template was made by Franklin Schamhart
